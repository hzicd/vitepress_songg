# 教程简明

## 初心

- 我看网上很多关于python编程语言教程无论是文档，还是视频，都是很繁琐的，不必要的废话很多，简单的东西给复杂化，有些东西听着很高大尚，各种术语名词，让初学者听了都头大。其实那些东西了解后，也就是那么回事。在这个躁动的社会，如果不是兴趣的驱使，一般人很难去快速了解上手一门编程语言。  
- 我将用大白话，且通俗易懂，文字少的中文，简要精炼提取重点，来写完这篇python编程语言教程。同时，也是为了做笔记，巩固一下知识点。
## 学习方法

- 如果你是对电脑零基础的纯纯小白，请先过一遍[计算机基础→](/Other/computer-basics/index.md)

- 学习很多东西，它往往不是以线性学习行得通的。尤其是计算机编程，一定是非线性了解学习，掌握的比较快。  

- 刚开始学习很多东西，你当时不理解，切记，不要死磕直接跳过。就像是去爬山一样，在山底，和山腰中，不知道庐山真面目，只有快速登顶，再回过头来，去看以前不理解的东西就迎刃而解了（毕竟我这套教程是面向于想要快速上手的小白）
