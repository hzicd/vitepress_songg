# 小毛病
写着篇log时候，还发现了之前的放弃的smoking目录，一直存在里面，忘记删了。居然还不影响打包，还有就是改了图片储存目录后，ide不会自动识别本地的图片了，需要手动敲图片的名字。我想可能是vitepress框架底层写的有对于插入的路径有规定，自动会找public目录，再去找后面img目录，所以只需要写/img/xxx.png了，这里的第一个`/`就是`public`,但是ide是从本地电脑出发的，才不会关心你是什么框架，他只认识正常规范的目录路径啊，他的底层框架是操作系统！！！
