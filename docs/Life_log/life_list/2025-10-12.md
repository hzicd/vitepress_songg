# 前言

起因是想去胖东来配个眼镜，其实说实在的，我目前的眼镜用着还行，除了容易脱落，其他方面还能接受。但是我还是想去换眼镜，我总感觉我目前的眼镜不好，他便宜，是在小县城配的眼镜，不好，不专业。

# 出发

下午两点起床了，什么也没吃，穿好衣服，洗把脸，背上书包就走了，走到寝室门口心里已经琢磨到，充电宝的线没带，又不想回屋了，但是带了又没任何鸟用，心里的斗争持续几秒中后，脚已经迈下楼梯这里了。最终还是没有回头，往楼下走了。其实，我不该带这个玩意。因为，我手机78格电足以让我可以用到。但是我就是这样的人，要把某件事的底线提高。也就是这个毛病，导致后面多个糟糕事。

# 坐错车&坐过车

也是挺马虎的，我很害怕马路，我每次过马路，神经几乎都是紧绷着，我很讨厌过马路，我有一种对马路的恐惧感，我害怕有一天我被撞死在马路上。我也很害怕开车，每次开车遇到一点不对劲的事情，赶紧紧急刹车。我生怕我死在马路上。我似乎有一种心理障碍的疾病，我自己称我这一现象叫“路恐”。走到学校南门，我要过马路了，坐公交车，我不知道我为什么当时没有想，应该坐学校这一侧的公交车站，我自然而然的过了我惧怕的马路，等马路对面的公交车。公交车开往3战后，我感觉不对劲，这么快到村里了，刚才那个十字路口没有左转，这时候我猛然意识到，我应该坐学校这一侧的公交车。我想过来的这一刻，刚好到一站了，我下车了，但是这时候我还不确定是不是做错了，但是直觉告诉我你就是做错了，双腿往回赶着，不过，还好，前面几百米的那一站是出发点，刚好也是这辆101号，没等多一会，就坐上了。高德地图告诉我37分钟就能到。这样过了2站，我又回到了我们学校，这里上来了几个校友。我直觉是他们估计也是去胖东来，但是我没当回事。我时会看手机，时会看着窗外发呆，脑子也没想什么，直觉告诉我要注意下车点了，感觉是快要到了，我想我的耳朵会告诉我的，因为他会听到“时代广场胖东来”这几个字的。校友他们几个下车了，但是耳朵他没听到这几个字，的确是没有听到，我也不知道。过了几站，我觉得不对劲，打开高德看下公交车路线，感觉是已经过了，但是我还是不确定。一个老奶奶站起来指了指说要去这胖东来要下车，司机告诉他前面就可以。我看到了胖东来，但是快到那个站牌，我看到了胖东来生活广场，我直觉告诉我错了，你应该去时代广场。我下车了。

# 糟糕的犹豫——致我的酷态科


这时候我是有另一个任务的，因为之前买的华莱士团购券0.01薅羊毛买的，可以白嫖一顿，不然17号就过期了，期间也不是周末。其实，说实话，我不该白嫖这一顿。我觉得这世间的事情就是如蝴蝶效应一样，一件事接着一件事看着毫无关系，其实有着微妙的关系。接下来，听我继续讲。我想先去华莱士把，走了100来米，又看了高德的时代广场胖东来，发现这不是同一个方向，还是在时代广场那里找一家把，又拐弯了，直达时代广场，这里很顺利我到了。凭我感觉负一楼是在中间下去，没错，对的。我下去了，看到了胖东来眼镜店，很小，我直觉告诉我，这里配眼镜和小县城配眼镜没有任何差别，还不如去大光明眼镜店。但是，我还是挂号了，免费的挂号，机器验了一下我的眼睛，然后等待第二个验光。我看了一下前面排了40来人，那个发的一张条有二维码，我扫一下就能知道，如果错过号，我将作废。我试着选了眼镜框，这里过了有个十来分钟吧，我想着要不要去其他地方转一转，我又扫码看了一下，还有40来人，只是少了2个人。我打开高德，看一下附近的华莱士，其实之前我就规划过，只是看高德地图觉得这有点远。当时在胖东来看了一下，才1公里左右，然后我就想要不要去，这40来人应该足够了，这进度这么慢。我做了一个错误的决定，我心里一直也在斗争。最终我的腿迈出了胖东来，开始前往华莱士，一路竞走，加上小跑终于到了，然后，拉开门，告诉他我是团购券，我以为要扫码，他给我说，到店自取就行，然后我下单了。这家卫生看起来并不好，2个员工看起来是一身的班味，椅子和座子都很破。旁边还有一个精神小妹在充手机，那个精神小妹，和这前台关系看起来很熟悉，他们时不时唠上几句嗑，前台边忙着手中的活，边给她说话，吐槽一些东西。然后我这是已经下单了，过了又7，8分钟，旁边那个男孩问101号好了没，好像说的就是101，然后就前台就给他了，这个前台刘海过眼带着鸭舌帽，我这时候回想应该能判断她是个女孩子，假小子。她走出前台，来到这女孩面前，面对面准备唠嗑。她看出我已经是点好的样子在等待，问我：点过了？？我回答了她，她那边没出票，她的直觉也是对的，她拿起手机，准备查我这一单，问我手机尾号，然后又问了那个后厨的男人，男人说我好像做过了，都很疑惑，为什么没有票呢，然后，假小子过来看了我的手机，看我点了什么，就直接做了，我大母手指，有意无意当着0.01的实付款。然后，就准备做我这一单，其实她看我手机的时候，外面又来了了一位中年妇女，假小子没搭理她。只是先紧着我这边看看是什么情况，我直觉告诉我这一家店并不好。假小子回头告诉了后厨，然后，开始做，这里过了3，4分钟，票出来了，有延迟。后厨男人说，应该把电脑砸了，买个新的，是不是酷狗音乐太占内存了，假小子说不是，就说电脑有毛病。他们扯着几句后，我这里一直在焦急等待，我怕我过号了，我扫了一下二维码，还有20人，我想如果还有15人我就要走，等会搞完眼镜再回来取也不晚。最后一直到了15人时候，又过了5，6分钟后，我等不了了，问了一句还要等多久啊？这时候精神小妹，往我这看了过来，我的余光告诉我，她拿起手机给我咔擦了一下。假小子没回答我，把鸡翅拿出来说：妈的，不炸了。我想快了，快了。假小子给我打包后，拿到前台桌子上面，精神小妹凑上前，嘀咕这什么，假小子说：什么什么？？你到底说的啥？精神小妹说等会你就知道了。我拿着袋子转身了准备走，悲剧在这里伏笔了，假小子说你还有一杯可乐，然后，我等他把可乐给我，我看到可乐是用机器封口的，以为不会漏掉，我把可乐放到了，这个含有汉堡鸡翅鸡块大袋子里面，一起装进书包。推开门，撒腿就跑。跑着有1分钟，我一直感觉我的后两个腿，怎么麻麻的，这种奇妙的感觉，我继续跑了10来秒，这种感觉又像是雨滴击打到我的大腿，直觉告诉我可乐撒了，我减缓慢下来，手摸了一下书包底部，湿的！我确定了，撒了。然后，我打开书包，拿出可乐，继续拿着它跑，直觉告诉我，你走错了，跑了有500米，你跑错方向了!!!!我赶紧打开高德确认，的确跑错了，然后，我又跑回去，我这时候已经意识到我的充电宝要不要拿出来，但是我心里又开始斗争说来不及了，赶紧跑回去吧，马上错号了，之所以我不拿出充电宝，是因为我感觉我已经把可乐放到大袋子里，漏了也不会波及到充电宝多少，其实我错了。我跑不动了，在路边，我害怕我被车撞死，我开始竞走，边打开手机，边扫码。还有12人，这时候我离胖东来回去路上还有7，8百米，我一直向天祈祷。这路上，我感受到成年人世间的无奈和悲苦。最后，走到门口后把手中的华莱士半瓶给扔掉了，路上我也没打量这个玩意，为什么会漏，口明明是封着的，我摇晃了几下，全是冰，好像只有1/5的可乐了。我感觉我脸通红，心跳急快。脸上全是汗，身体闷热，想脱掉外套。最终，我到了眼镜店是还有9个人，我走向了厕所，书包里的卫生纸都因为这瓶可乐废掉了，我看到充电宝3个口里面全是可乐，我意识到坏了。我把那半湿掉的卫生纸拿出来给我的充电宝擦拭，又猛甩充电宝，试图利用惯性把可乐甩出来，是甩出来一点。但是我当时觉得，后面晒干一下应该还能用，把它放到外面这个包里面。其实我错了，最后回到学校时候发现已经打不开机了，按下电源键没反应，我100%坚定我前天冲到88电量就放那里了。我也不敢充电，先放到阳台吧，但是河南这半个月的梅雨没停过，一直都是阴潮的。

# 我真的需要换眼镜？

说实在的，我可以将就，或者是说，根本完全不需要配镜，我左眼是450，散光100，右眼450，散光125。也就是40个人后，排到我了，那个验光师，测了我现戴的眼镜度数是差不多的，就仅仅是右边的眼涨了25度而已，她建议我完全不需要配镜，如果你的需求是想看清的话。其实，第一轮机器验光后，我就已经感觉到胖东来的配镜流程跟我们小县城的小店一模一样，有的还不如我们那些大光明的店，因为在大光明配镜的时候，给我配镜检测流程非常多，我也去了解过，大概这些测试都是在测什么。我去胖东来就是想着他那边能不能把散光的方向给测出来。想要针对我散光方面专门测一下。结果并没有，自己看了一下电子视力表就行了，那个验光师先让我试戴10分钟左右回来。其实胖东来的这一系列的测试都是免费的，我就听一个中年男人调侃说，免费的验光有人不觉得这里不踏实，什么的。的确啊，这种验光流程，基本在小店里面，也是这样走的，非常简单。10来分钟我回来了，我告诉她我的需求在于这个散光上面（尤其看电棒式灯重影严重），能不能恢复矫正呢？她说你的度数已经调到最好了，这个跟你的用眼习惯，要多出去，不要老是看电子产品。其实我不建议配镜，她三番五次说我。我给她说了一些其他需求，就是眼眶爱掉了，镜片磨的有一点花了。然后，最后我坚定告诉她我还是要换镜，他给我开了条，让我找那个阿姨看一下选镜框和镜片。我过去告诉阿姨镜框我已经选好了，她让我拿过去给她看看，然后就是选镜片了，她说你是想要好一点的，还是性价比的，我说性价比吧。其实，这时候我心里是知道我是要蔡司或者依视路的镜片的，这算是顶尖的了。我犹豫了3秒说性价比吧，他给我说有这个月亮的什么的400多，我都没在意听了。我直接问她蔡司的呢，她说蔡司1.6，一千二百多，我又问她依视路呢？比蔡司便宜一点，一千一，依视路要10天左右，需要进货预定，蔡司15号就可以拿了。我想了想，其实，我还是想选依视路，不知道为什么。最后，我说要蔡司1.6吧，她说要蔡司是吧，行。她拿出计算器敲了记一下，嘴里说着蔡司2000多打6折，加上镜框209,一共是1397。我对这个价格其实并不惊奇，其实我已经预算出血的话1500来配眼镜了。这样看来，我自己觉得就有点败家，在没那个急需和必须的角度来看。最后，在付款处付了钱，然后，把付款条凭证条给了那个阿姨，最后告诉我15号来拿就行了。我就走了。



# 回去&感想

走出胖东来，天已经是黑透的，我找了一个凳子坐下，打开书包拿出华莱士汉堡，包装纸已经被可乐浸透，我拨开上面的湿纸皮，带上我的卫衣帽子，晚上的风刮的很是厉害。我不由得感想，这1000来块是否花的值得，我觉得并没有，我的目标地是这里能解决我的散光问题，我可以出血到1500，结果不尽人意吧。这一天的遭遇，为什么如此狼狈，吃着手中的华莱士，没有可乐，快要把我咽死。我吃完后，鸡块和鸡翅打算回去再吃。我再一次打开高德地图，边走边查公交车，看到路上有几个，我想这还没下班，已经快了，高德上标识说6:45是末班车，我可以做Y3路，和101路。101在下面，我没注意，以为还要转车到101。我在胖东来这个交通等了一会，一个车过去了，紧接着101来了，我不知道要不要坐上去，我还想要在仔细查一下，我才心安理得，但是直觉告诉我等不了，我直接上了这个车，看到这个路线是有我们学校的，再看一下移动方向，又想了想来的时候也是101。我这时候断定我没坐错。但是过了几路，我直觉告诉我这个司机，开车不好，有点像路怒一样嘟囔几句那些“不长眼”的车。后面，几站整个车都是颠颠晃晃的，搞的我心里难受的想吐，一种瘆人的感觉。终于，到了学校，我趁着领了快递，进入餐厅买了瓶2.5的可乐，坐在餐厅准备把鸡翅和鸡块吃掉。吃到鸡翅的时候发现需要，滤到骨头，感到有些不雅。嘴里含着，走出餐厅，吐到垃圾车里面。回到寝室了，吃掉了这些。不过，现在，我在记录这篇日记的时候，我的肾结石发作了，我不知道是，在路上跑回胖东来的原因，小石头给晃动位置了，还是我吃了这华莱士的原因，导致我这两边非常难受。我在我之前的日记，也有写到这个小毛病，我只是直觉告诉我这是肾结石导致的。之前我得过肾结石，把大石子给震碎了。后来，我肾里还有小石子，因为太小，医生说不能用超声波来震，根本震不到，而且伤害肾。
