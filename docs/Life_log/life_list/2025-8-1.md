# 前言
我在7月19号下午入职了，家里楼下200米处的奶茶店当暑假工。老板是个有钱人，在我的镇上有一套小区，我的很多亲戚都卖了他家的房子。按辈分30岁的老板应该喊20岁的我 `表叔`。好像是我爷爷的母亲的亲妹妹的子子孙孙，老板多多少少也能给我扯上点亲戚关系。然而，在8月1号下午6点我跑路了。

# 感悟之苦
其实，我不再想要记录吐槽我这10来天怎么过来的，这里说一下感悟吧！我曾经以为挣钱，单纯能吃苦就能挣到钱。其实，苦还分为肉体上的苦，和精神上的苦。如果排除那些极端的累死人的巨消耗体力的离谱工作。对于一个健康的正常人来说，肉体的苦，身体上的劳累都不算什么，他都能忍受，都能耐得住，有时候忍忍就过去了，甚至对他来说这些皮肉之苦，看来显得不堪一击。但是，还一种苦，是精神上的折磨，比如，老板的刁难恶心，同事之间的人际关系。一般这种直击精神上的苦，能吃的人很少。因为，相对于身体劳累的皮肉之苦，不也最后转换为神经信号，传给你的大脑，告诉你，身体非常累。起码，这种是间接的。那种精神上的直接冲击，直击你的灵魂。有时候身体都反应不过来。真的会损伤身体器官，影响身心健康，甚至精神萎靡。可能听起来，也没什么，甚至觉得自己什么东西扛不住？其实，真的轮到你头上，真的去遭遇，各种各样的傻逼的人，各种烦恼恶心事，那时候就真的老实了。
# 真的是你吗？命运
我是一个挺相信命运的，我一直不明白每个人，甚至人类的根源在哪。这一系列的蝴蝶效应的发生，潜移默化的走完了每一个人的命运。

# 关于记录
其实，有很多东西，我都想详细的给记录下来，但是不知道为什么，我很懒。只在脑子里有东西，不想打字记录，当我去寻找这些东西去用文字记录的时候，往往又变了，或者是说模糊消失了。可能一些细节东西不能被记录，只能存在我的脑神经里。这种现象不过也是命运的一种。
