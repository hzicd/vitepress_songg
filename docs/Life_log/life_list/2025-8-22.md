# 前言  
这几天我把王者注销卸载了，决定再也不碰这个破游戏了。其实之前也注销过很多次，但是都因为某些原因又给下回来了。这次是一定不会玩了。然后，就是整天开始刷抖音，看b站峰哥。由于我已经有一些不怎么玩游戏的倾向了在这之前。刷视频的时候，抖音的算法机制，也很少给我推荐王者，直播啥的。我一般刷到也忍不住看上几秒，然后就划走了。但是，抖音给我推了一些关于车的视频，其实，我对车并不感兴趣。我的兴趣还是在数码这块吧。  

# 关于车

我不知道抖音，为啥给我推了几个车祸视频，我越看，它就越推。可能是，我之前考过驾照，但是我一直没敢去真正的开车上路。其实，考完那个驾照，我心里打底就知道，有没有这个驾照，我一样不会开车。我知道开车，这玩意还是要多实战，多上路才能开。拿个本本不过是糊弄交警，而且考驾照，对于我来说，它只是告诉了你汽车最基本的的东西。虽然，家里有一辆五菱宏光的面包车，但是我也懒得去练。因为，我对车，对户外根本没啥兴趣！！！为啥我要这样说呢？刷抖音的时候，由于我看了好几个车的视频，抖音算法开始发力了，一直给我推车，有一个女司机，天天拍vlog记录她每天开车去上班，买菜的视频，她也是女司机，开的BYD的。我也不知道具体什么型号了，因为我对车这块根本是啥也不知道。当时，我就给她点上关注了，有个7，8个作品吧，她就有2.1w粉丝。的确，我很敬佩女司机，而且，我特别欣赏，甚至仰慕那些开车的女司机。后面抖音又给我推送了一些，富二代女司机。不是，那种显贵败家的。而是，给人一种，成熟，有责任，有能力，有见识，有胆魄，看着经验丰富的女司机。而且，外貌还很漂亮，像是御姐型的。这一下子引发了我对人生的思考，其实，以前我也会经常会有这样的人生思考，只是没记录下来。因为语言表达不利索，还有就是懒得记。

# 感悟

我突然就是非常不解，为什么我们生活有着天壤之别。但是，我又想从这里面，找出我们客观公平的地方，我去思考到底是什么导致的，为什么会这样？相同的地方和不同的地方都是为何导致。我很难用语言去表达清楚，我尽量说的明白。首先说为什么我对车这么不感兴趣，而且他们为什么那么早就开始懂车了。我觉得是原生家庭环境带来的。这个不必多说，都知道的。但是，在这原生家庭不同之处，同时有着相同的东西，那就是，我们都会抱怨原生家庭的不满，因为，无论是穷哥们，还是富哥们，原生家庭不是完美的，他总会给你带来烦恼。但是能带来财力，资源是不同的，这一点我们必须肯定，也就是因为这一点，别人从小在外国留学，那些见识和观点，以及环境等众多因素影响下，催生出的个人兴趣爱好的不同。也就是到后来，我们通过互联网，偶然间，彼此看到对方，穷哥们是带着一副厚重的高度近视眼镜，敲着键盘的穷b，另一个则是有着一头时尚卷毛，单手打方向盘坐在跑车里的富哥们。我想说的是啥呢，为什么说着穷哥们带着厚重高度近视的眼镜，追根溯源穷哥们的成长历程，可能是留守儿童，用着父母留下的3手旧的手机，天天在别人墙角下蹭网，能卡成PPT，还天天玩不够尽兴，上学的时候，无脑听了老师父母的话，起码是一个有良心的中国人，大多穷哥们都会好好学习，拼命看书。因为在中国从小被灌输儒家经典学说，深深刻在他的脑子里，他不得不刻苦学习。再去想象富哥们，成长历程那时候他们在干什么，他们可能有着丰富的玩耍，旅游历程，父母有很明智的正确引导，而不是，天天跟穷哥们一样，天天玩ppt游戏，因为那些玩意，他已经玩够了。这就是不同环境等众多因素，催生出的性格，语言表达，兴趣爱好的不同。就这3点足以，可以让着2类人放到不同世界里生活。可就是，互联网让一方穷哥们窥探到了富哥们的生活。这差别就是就是不同的前因后果啊！让人觉得，根本不是一个世界的人，但是在使用语言上都是汉语，有时候穷哥们的长相外貌，还出众于一些歪瓜裂枣的富哥，穷哥们或许会感一些温馨。

# 破局

尽管啊！我们表面上是生活在巨大的差距里，但是我认为人一辈子在精神层面上分泌的多巴胺也好，内啡肽也罢，在某种玄学上说是平等的。所以啊，就是干自己喜欢的事情吧！因为，穷哥们的兴趣爱好和富哥们兴趣爱好的不同，他们的天壤之别是前因，环境等一切复杂因素造就的。他们都在干自己在自己认知范围内，让自己踏实，快乐，舒服的事情。或许，穷哥们，有一天，会有着一番好的事业。也许，富哥们，也在为自己喜欢的事业付出，最后都付之东流。也有可能，会比穷哥们更牛逼，穷哥们也未必能闯出一片天。
