# 前言

看完b站up主数字游牧人的视频后，有些感受。又引发了我对人生的思考。

# 自我阐述？

其实我很早就看过数字游牧人的视频，并且关注了他。当时只是感觉他的经历很牛比。我很喜欢听别人的经历故事，因为我觉得我自己的人生经历，截止目前20岁了，没什么可拿出来说的东西，平平无奇，普普通通。说话表达也是语无伦次，没有逻辑性，至少我这样感觉，在记录一些东西的时候总感觉写的一些话，调换一下其中的词语的位置，表达可能会更好，但是我敲上之后，就不想去修改了。我很欣赏up主的人生经历，我在网上了解过很多人写的博客，短视频分享的自己的人生经历，我都很有耐心去听，看下去。我是个很躁的人，我没有性子去看长篇的文章，根本没有任何耐心去读任何一本书，我天生就是看见稍微长篇的文字就想要把这张内容给毁掉的人，我看不进去文字多的东西。就像是高中数学题的一些最后一道大题，其实第一问非常非常简单，但是题目给的很长，我没有耐心看下去，很少得过分。我不理解为什么我是这样的。我似乎尝试改变，但无济于事。而且，我对历史非常鄙视，我更恶心看这种长篇大论的书籍。
# 差距的起源？

我想了很多，我知道人生经历这个东西会塑造两个天壤之别的人。可那个人为什么不是我？我这个不明白，我也不理解。为什么，他们口中的一些观点说法，说自己是个普通人，听似很简单。实则不然，这是家庭一代一代积累，迭代的结果。我觉得父母，能给下一代的影响占了90%，后面只是一系列的蝴蝶效应罢了。大多是父母，当然有个例是孤儿，这要看自己遇到什么事情和人了。不过也是一系列的蝴蝶效应罢了。举个简单例子：很多牛逼的人说自己年薪百万，一个月能整2个w，说自己是普通人，年龄20左右，已经游走好几个国家。看看他们的父母是什么人？当然他们会说自己父母最多是个体制内的普通人，一个月拿个6k工作算顶天了，似乎和你的父母拿着4000多工资差不了哪去。你可曾知道那些父母的认知，比你父母高出好几个点。就一个交流，很多底层人跟父母都是没办法交流的，我不知道为什么会有这道鸿沟。而那些所谓自己是普通人的没有这道鸿沟。跟自己父母交流起跟朋友一样。这就是一代一代的迭代。等到你这一代当父母，再去抚养你的下一代，你的格局似乎到了，那些自称年薪百万的普通人父母的半个点。可人家孩子又是一代的进步。这其中差距到底是从何而起的。我不知道，我能想象的也就是可能是，以前沿海城市的人，率先接触外面的世界，率先领先内陆把。在古代，可能是吃的多，有力气干活的人，率先努力奋斗的人，跟普通猿人类拉开差距的原因吧。如果这一切就按照这个节奏，没有外界灾难威胁，差距就这样诞生出来了吧。这是我的想象，我也只能想到这了。

# 我为什么是我？
我不理解，那个人生经历无比丰富的那个人为什么不是我？为什么我来到这个世界上，过这种索然无味的苦逼生活，我为什么不是皇帝，我为什么投胎到河南当牛马，我为什么不是富二代，为什么不年薪百万。为什么我就是我？我不知道，有人会说你可以去改变啊，可是我想问你的是：能有改变这个想法的我，为什么不是我，而是没有有着改变心态，没有着上进心去改变的那个人？为什么我就是我？我不理解，我为什么投胎过来？
# 全局观？
无论我接触任何事情，视频也好，身边的人也罢。我都会以上帝的视角区观察，去思考这种变化的源头，类似蝴蝶效应，是怎么引发的。就好像是营销号在给你说一些八卦时候，我10%精力关注他讲什么，90%在想他为什么做这个账号，再去看看评论区，看看其他人观点，点开他的主页，分析为什么他会评论这样的话，看他的ip，潦草看一下他的作品。尝试分析，为什么他会说这句话。而不是我眼前的信息的刺激。在跟别人交流，我会过头经常想，他可能会有怎么样的人生经历，他的家庭，或者什么因素也好，怎么会导致这个人这样的性格，说这句话目的何在？更多的是换位思考的心理学，我更喜欢上帝视角区洞察这一切！

# 关于记录
我一直没有记录东西的习惯，拍照，日记也好。对这个事情进行上帝视角分析：我高3以前一直没有回忆念旧的习惯，从没有，或者说通过记录的一些东西，念旧去享受回忆我从来是没想过有这个东西存在！可是我现在在记录东西了，我也就知道了，爱记录的人可能心里和经历。以前我是从来不记录东西的，一直活在当下，那时候脑子不灵活，傻了吧唧的。也就是说，很早能意识到记录东西通过这些记录回忆东西的人来说，脑子都是比较成熟快的。但遗憾的是，从他爱记录的那一刻，他便没有以前快乐了，也就是说脑子成熟快的人，先接受的痛苦比傻子多！因为傻子都是活在当下，而我当了20年的傻子！
