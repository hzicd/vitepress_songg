# 计划打破
昨天我说过，如果在家躺两个月的话，我会学一些编程来打发时间。当然，也可以打游戏，做任何事情来打发。因为，我是一个比较宅的人，社交我也挺喜欢的，但是我懒得去。我原本是计划好的，一个月把linux基础学完，第二个月，一些基础爬虫和mysql+其他杂识。就是打算第二个月爆肝自己了。结果，来奶茶店上班了，昨天下午入职，还想着后面能早起，就先把linux慢慢学，晚上下班再学一点。进度干上就行。正当我以为这个月末，就开始我的爆肝行动了。但是，我的老毛病又犯了，我一直没找到这个老毛病病原在哪，一旦发作就是什么也想不进去了。如果没有这个毛病，我爆肝自己，还是能顶着住的。就今天早晨，6点多就睡醒了，就先过一半的linux，（一天进度的一半），打算晚上下班，再过后面的。我靠，那个臭毛病发作了，直接就是脑子瘫痪，不能深入搞进去任何学习内容。我找的linux课程，在b站上，视频都有标签进度，刚好一个月。我想我停在了20day，后面的课程只能到开学后面看看了。
#  人无完人
人其实并不完美，或者说世界所以东西都一个相对性，都有一个平衡点，你某个方面有极端，必然在某个不知处有另一个极端。其实，我身上这种小毛病都是我高三得的，虽说不影响正常生活，但是会恶心正常生活，就像有天上有口痰，时不时精准吐到你脸上，也没任何办法解决它。这个情绪都低落了。这是为什么？？？为什么要这样惩罚我？就是不想干任何时期，就是隐痛的难受。也睡不着，小病魔为什么总是缠我身上？主啊，怜悯一下底层人吧！
